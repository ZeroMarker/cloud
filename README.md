# cloud

Common CLI commands for cloud providers and VPS platforms.

Pricing, plan, and free-tier notes: `pricing-free-tier.md`

## Providers

- Alibaba Cloud: `aliyun/aliyun.sh`
- Amazon Web Services: `aws/aws.sh`
- Microsoft Azure: `azure/azure.sh`
- Google Cloud Platform: `gcp/gcloud.sh`
- Tencent Cloud: `tencent/tencentcloud.sh`
- Huawei Cloud: `huawei/hcloud.sh`
- Oracle Cloud Infrastructure: `oracle/oci.sh`
- IBM Cloud: `ibm/ibmcloud.sh`
- DigitalOcean: `digitalocean/doctl.sh`
- Vultr: `vultr/vultr.sh`
- Linode / Akamai Cloud: `linode/linode-cli.sh`
- Hetzner Cloud: `hetzner/hcloud.sh`
- Contabo (low-cost VPS): `contabo/cntb.sh`
- IONOS (managed-friendly): `ionos/ionosctl.sh`
- Hostwinds (US network): `hostwinds/hostwinds.sh`
- Kamatera (flexible scaling): `kamatera/kamatera.sh`
